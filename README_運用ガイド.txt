============================================
  基本の使い方
============================================

【社員（毎日）】
  1. スマホでアプリURLを開く
  2. 自分の名前をタップ
  3. 日付・現場名・出退勤時間・備考を入力
  4. 「保存する」をタップ

【管理者（月末）】
  1. PCでアプリURLを開く
  2. 「管理画面」をクリック
  3. 社員・月をフィルター選択
  4. 「Excel(.xlsx) 出力」でダウンロード
  5. 給与計算に使用


============================================
  勤務時間の計算ルール
============================================

  定時: 8:00〜17:30
  休憩: 90分（15分+15分+昼60分）
  所定労働時間: 8時間

  8:00より前に出勤 → 早出（例: 7:30出勤 = 0.5H）
  17:30より後に退勤 → 残業（例: 18:30退勤 = 1H）
  日をまたぐ勤務にも対応（例: 8:00〜翌2:30）

  ※設定は管理画面の「設定」タブから変更可能


============================================
  スマホでアプリ化する方法
============================================

  【Android】Chrome右上の︙ → 「ホーム画面に追加」

============================================
  主要ファイル一覧
============================================

  server/index.js    ... APIサーバー・計算ロジック
  server/db.js       ... データベース初期化
  server/excel.js    ... Excel出力
  client/src/components/
    LoginScreen.jsx  ... ログイン画面
    InputScreen.jsx  ... 勤務入力画面
    ListScreen.jsx   ... 勤務履歴・月次サマリー
    AdminScreen.jsx  ... 管理画面
  client/src/api.js  ... API通信
  client/src/utils/time.js ... 時間計算


============================================
  注意点
============================================

  ・無料プランはしばらく使わないとスリープする
    → 最初のアクセスに50秒ほどかかることがある
    → 自動リトライ機能あり

  ・デプロイし直すとDBがリセットされる可能性あり
    → 月末にExcel出力してバックアップ推奨

  ・本格運用なら月額7ドルの有料プランでデータ永続化可能
